# Explanation topic

--8<-- "docs/topic-types/single-source.md"

 The purpose of an explanation topic is to describe the at a high level the concepts relevant to the package or project. It is focused on understanding and providing context. Explanation topics are the only topics that make sense to read while away from the product itself. Avoid duplicating content that belongs in tutorials or reference topics and link to those topics instead. For additional information on how to structure explanations, see [Diataxis](https://diataxis.fr/explanation/).
